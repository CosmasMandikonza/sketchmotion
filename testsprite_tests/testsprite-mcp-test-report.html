SketchMotion baseline report

Issues that mattered

Why this file is intentionally simple

The durable submission artifacts are the checked-in tests plus the written QA documents: BUG_FIX_LOG.md and TESTING_STRATEGY.md.

This baseline report is preserved because it shows that SketchMotion was already behaving like a real web application before the second hardening round expanded into failure handling, invalid states, persistence, and GLM-specific coverage.
